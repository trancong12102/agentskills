# Agent Skills

A collection of reusable agent skills for AI coding assistants.

## Skills

| Name | Description |
|------|-------------|
| [conventional-commit](skills/conventional-commit/SKILL.md) | Generates commit messages following Conventional Commits 1.0.0 specification |

## Usage

Copy the desired skill's `SKILL.md` file to your agent's skills directory.
